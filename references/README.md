# References

This folder holds section design screenshots for future reference.

Screenshots are used to:
- Document the intended visual design of each section
- Provide before/after context when redesigning sections
- Give handoff reference to developers or collaborators who didn't work on the initial build

---

## Naming Convention

Screenshots should be named by page and section:

```
{page}-{section}.png

Examples:
  home-hero.png
  home-values.png
  home-services.png
  home-stats.png
  about-intro.png
  contact-form.png
```
